# 디지털 망원경 키오스크 UI 구현 계획

## 1. 개요

전망대에 설치되는 디지털 망원경 키오스크의 UI 프로토타입을 QML + PySide6로 구현한다.

### 제품 특징
- 40배율 광학줌 카메라로 풍경을 실시간 관람
- AI 기반 랜드마크 사물인식 및 정보 표시
- 신용카드 결제 후 3분간 관람 가능
- 26인치 터치스크린
- 키오스크 기능: 광고, 공지, 명소 안내

### 이번 구현 범위
- **포함**: UI 화면 디자인, 화면 전환, 시각 효과, 애니메이션
- **제외**: 결제 백엔드, 카메라 연동, AI 인식 로직, 하드웨어 연동

### 기술 스택
- PySide6 (Qt for Python)
- QML (Qt Quick)
- Windows PC 프로토타이핑 (추후 Jetson Orin Nano 이식)

## 2. 화면 흐름

```
                    +--[취소]--+
                    |          |
[홈화면] --관람시작--> [결제화면] --결제--> [관람화면] --3분경과--> [종료화면] --4초--> [홈화면]
                                           |      ^
                                           |      |
                                           +------+
                                         [명소상세패널]
                                       (오버레이, 탭으로 열기/닫기)
```

### 화면 목록
| 번호 | 화면 | 설명 |
|------|------|------|
| 1 | HomeScreen | 홈/광고/슬라이드쇼 |
| 2 | PaymentScreen | 결제 안내 |
| 3 | ViewingScreen | 관람 (카메라 피드 + AI 오버레이 + 타이머) |
| 4 | LandmarkInfoPanel | 명소 상세 (관람화면 위 슬라이드인 패널) |
| 5 | SessionEndScreen | 관람 종료 감사 메시지 |

## 3. 디자인 시스템

### 컨셉: "Premium Observatory"
야외 전망대에서 사용하는 고급 망원경의 느낌.
어두운 배경 위에 깔끔한 타이포그래피, 글래스모피즘(frosted glass) 패널, 부드러운 애니메이션.

### 색상 팔레트

| 용도 | 색상 | 코드 |
|------|------|------|
| 배경 | 깊은 다크 | `#0D1117` |
| 표면 (글래스) | 반투명 흰색 | `rgba(255,255,255,0.08)` |
| 글래스 보더 | 반투명 흰색 | `rgba(255,255,255,0.15)` |
| 주요색 | 블루 | `#3B82F6` |
| 강조색 | 앰버/골드 | `#F59E0B` |
| CTA 그라데이션 | 블루-퍼플 | `#3B82F6` -> `#8B5CF6` |
| 텍스트 (주) | 밝은 흰색 | `#F8FAFC` |
| 텍스트 (보조) | 회색 | `#94A3B8` |
| 성공 | 에메랄드 | `#10B981` |
| 오류 | 빨강 | `#EF4444` |
| 경고 | 앰버 | `#F59E0B` |

### 타이포그래피

| 용도 | 크기 | 굵기 |
|------|------|------|
| 히어로 텍스트 | 72px | Bold |
| 가격/카운트다운 | 48px | Bold |
| 섹션 제목 | 28px | Medium |
| 본문 | 20px | Regular |
| 캡션 | 16px | Regular |

폰트: **Noto Sans KR** (Google Fonts)

### 시각 효과

| 효과 | 설명 |
|------|------|
| 글래스모피즘 | 반투명 배경 + 블러 + 미세 보더 |
| 그라데이션 오버레이 | 이미지 위 하단->상단 페이드 |
| 화면 전환 | 300-500ms easing 슬라이드/페이드 |
| 터치 피드백 | press 시 scale(0.96) + 밝기 변화 |
| 카운트다운 | Canvas 원형 아크 프로그레스 |
| 슬라이드쇼 | 1초 크로스페이드 전환 |
| 바운딩박스 | 글로우 보더 + 반투명 배경 |
| 경고 펄스 | 30초 이하 시 빨간색 깜빡임 |

### 터치 규격
- 버튼 최소 높이: 80px
- 버튼 라운드: 16px
- 아이콘 크기: 48px
- 여백 대: 40px / 중: 24px / 소: 12px

## 4. 화면별 상세 디자인

### 4.1 HomeScreen (홈화면)

**레이아웃:**
```
+--------------------------------------------------+
| [위치명]                    [날짜/시간]  <- StatusBar (반투명)
|                                                    |
|           (전체화면 슬라이드쇼)                      |
|           풍경 사진 자동 전환 (5초)                   |
|                                                    |
|~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ |
|  그라데이션 오버레이 (하단에서 올라오는 어둡기)        |
|                                                    |
|  "전망대 디지털 망원경"  <- 히어로 텍스트             |
|  "아름다운 풍경을 40배율로 감상하세요"                |
|                                                    |
|  [남산타워] [롯데타워] [한강대교] [...]  <- 명소 카드  |
|  (가로 스크롤, 글래스 패널, 썸네일+이름+거리)         |
|                                                    |
|        [======= 관 람  시 작 =======]  <- CTA 버튼  |
|           (그라데이션, 대형, 풀폭)                    |
|                                                    |
|  --- 공지: 운영시간 09:00~18:00 ---  <- 배너        |
+--------------------------------------------------+
```

**핵심 요소:**
- ImageSlideshow: 2개 Image를 교대로 OpacityAnimator 크로스페이드
- LocationCard: 글래스 패널 안에 썸네일 이미지 + 명칭 + 거리 표시
- 하단 그라데이션: LinearGradient (transparent -> #0D1117)
- CTA 버튼: LinearGradient 배경 (#3B82F6 -> #8B5CF6)

|         |   | 대어주세요          |   |            |

### 4.3 ViewingScreen (관람화면)

**레이아웃:**
```
+--------------------------------------------------+
|                                         [2:45]    |
|                                        (원형 타이머)|
|                                                    |
|      +--- AI 인식 ---+                   [+]      |
|      | 남산타워       |                   [ ]  줌  |
|      +---------------+                   [-]      |
|                                                    |
|              (전체화면 풍경 이미지)                   |
|                                                    |
|                    +--- AI 인식 ---+               |
|                    | 롯데타워       |               |
|                    +---------------+               |
|                                                    |
|  +--- AI 인식 ---+                                 |
|  | 한강대교       |                                 |
|  +---------------+                                 |
|                                                    |
| [30초 이하 경고: "관람 시간이 곧 종료됩니다"]         |
+--------------------------------------------------+
```

**핵심 요소:**
- 배경: 전체화면 풍경 이미지 (카메라 대용)
- CountdownTimer: Canvas로 원형 아크 그리기 + 중앙 시간 텍스트
  - 정상: 흰색/블루
  - 30초 이하: 빨간색 + SequentialAnimation 펄스
- DetectionBox: 점선/글로우 보더 Rectangle + 상단 라벨
  - 탭 시 LandmarkInfoPanel 슬라이드인
- ZoomControls: 세로 배치 +/- 원형 버튼

## 5. 프로젝트 구조

```
kiosk/
  main.py                           # PySide6 앱 진입점
  
  qml/
    Main.qml                        # ApplicationWindow + StackView
    Theme.qml                       # 디자인 시스템 싱글턴
    
    screens/
      HomeScreen.qml                # 홈화면
      PaymentScreen.qml             # 결제화면
      ViewingScreen.qml             # 관람화면
      SessionEndScreen.qml          # 종료화면
    
    components/
      ImageSlideshow.qml            # 크로스페이드 슬라이드쇼
      LocationCard.qml              # 명소 카드
      TouchButton.qml               # 터치 버튼 (ripple/scale 효과)
      GlassPanel.qml                # 글래스모피즘 패널
      StatusBar.qml                 # 상단 날짜/시간/위치
      AnnouncementBanner.qml        # 공지 배너
      CountdownTimer.qml            # 원형 카운트다운 (Canvas)
      ZoomControls.qml              # 줌 +/- 버튼
      DetectionBox.qml              # AI 바운딩박스 + 라벨
      LandmarkInfoPanel.qml         # 명소 상세 슬라이드 패널
  
  assets/
    images/                         # placeholder 이미지
    fonts/                          # Noto Sans KR
```

## 6. 구현 단계

### Phase 1: 기반 + 홈화면
1. `main.py` 작성 (PySide6 앱, QML 로드)
2. `Theme.qml` 디자인 토큰 정의
3. 공통 컴포넌트: TouchButton, GlassPanel, StatusBar
4. `Main.qml`: StackView 셸
5. `HomeScreen.qml`: 슬라이드쇼 + 명소카드 + CTA + 배너

### Phase 2: 나머지 화면
1. `PaymentScreen.qml`: 결제 안내 UI
2. `ViewingScreen.qml`: 관람 화면 (풍경 + 오버레이 + 타이머)
3. `CountdownTimer.qml`, `ZoomControls.qml`, `DetectionBox.qml`
4. `LandmarkInfoPanel.qml`: 명소 상세 패널
5. `SessionEndScreen.qml`: 종료 화면

### Phase 3: 연결 + 폴리시
1. 화면 간 네비게이션 연결 (StackView push/pop/replace)
2. 카운트다운 타이머 동작 (QML Timer)
3. 전환 애니메이션 적용
4. 전체 비주얼 검수

## 8. 실행 및 검증

```bash
cd kiosk
python main.py
```

검증 항목:
- 전체 화면 순회: 홈 -> 결제 -> 관람 -> (명소상세) -> 종료 -> 홈
- 슬라이드쇼 자동 전환
- 카운트다운 타이머 동작 및 경고 표시
- 버튼 터치 피드백 애니메이션
- 화면 전환 애니메이션 부드러움
- 글래스모피즘 효과 렌더링

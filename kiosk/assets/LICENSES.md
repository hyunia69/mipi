# Asset Licenses

All assets in this directory are bundled for commercial kiosk use. Each entry
records the upstream source, author, and license so the kiosk can be shipped
without legal risk.

The authoritative, machine-readable record is `assets/metadata.json`, which is
generated by `scripts/fetch_assets.py`. This file is a human-readable summary.

## Fonts

- **Pretendard** — <https://github.com/orioncactus/pretendard>
  - License: SIL Open Font License 1.1 (OFL-1.1)
  - Usage: primary UI font (Korean + Latin)
  - Files: `assets/fonts/Pretendard-*.otf`

Fallback fonts used by the application if Pretendard is not installed:
- Malgun Gothic (system-bundled on Windows, Microsoft)
- Noto Sans KR (system-bundled on Jetson, Google, OFL-1.1)

## Icons

- **Lucide** — <https://lucide.dev>
  - License: ISC
  - Usage: UI icons (zoom controls, status bar, info fields)
  - Files: `assets/icons/*.svg`

## Landmark Photography

Photographs of Seoul landmarks are fetched from the following sources by
`scripts/fetch_assets.py`. Each photo's exact author and source URL is captured
in `assets/metadata.json`.

- **Unsplash** — <https://unsplash.com/license>
  - License: Unsplash License (free for commercial, no attribution required,
    but attribution is recorded for goodwill)
- **Pexels** — <https://www.pexels.com/license/>
  - License: Pexels License (free for commercial, no attribution required)
- **Wikimedia Commons** — <https://commons.wikimedia.org/>
  - License: per-file (CC0 / CC-BY / CC-BY-SA). Only CC0 and CC-BY files
    matching the manifest allowlist are downloaded.

## Trademark Note

Building exteriors (Namsan Tower, Lotte World Tower, etc.) are photographed
as public works of architecture. No corporate logos or trademarked graphics
are embedded in the kiosk UI. Avoid using photographs that prominently feature
branded signage.
